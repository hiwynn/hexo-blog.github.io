lodashjs —— Array（一）

发表于 2019-04-25 | 分类于 代码

_.chunk(array, [size=1])

创建一个有若干个元素组成的数组，这些元素由原数组按照 size 的长度分割而来。如果数组不能被平均分割，最后一个元素会被单独分割出来。

_.compact(array)

创建一个移除了所有假值的数组。假值包含：false, null, 0, “”, undefined, 和 NaN

_.concat(array, [values])

创建一个由该数组和（或）其他数组或值组合在一起的数组。

_.difference(array, [values])

使用 SameValueZero 创建不包含在其他给定数组中的数组值数组，用于相等比较。结果值的顺序和引用由第一个数组决定。

与 _.pullAll 不同，这个方法返回一个新的数组。

参数

  • array (Array): 要检查的数组
  • [values] (…Array): 要排除的值

返回

  • (Array): 返回新的过滤后的数组

_.differenceBy(array, [values], [iteratee=_.identity])

这个方法和 _.difference 类似，不同的是它接收了迭代器并被数组的每个元素和值调用，以生成用于比较他们的标准。

与 _.pullAllBy 不同，这个方法返回一个新的数组。

_.differenceWith(array, [values], [comparator])

这个方法和 _.difference 类似，不同的是它接收了比较器并比较被数组的每个元素和值。

与 _.pullAllWith 不同，这个方法返回一个新的数组。

_.drop(array, [n=1])

创建一个从数组的开始删除 n 个元素的数组，默认 n 为 1。

_.dropRightWhile(array, [predicate=_.identity])

创建一个数组切片，其中不包括从末尾删除的元素。元素会被删除直到 predicate 的返回为 falsey（不再继续迭代）。predicate 有 3 个参数：value，index，array。

原数组不会被改变。

参数

  • array (Array): 要查询的数组
  • [values] (…Array): 每次迭代调用的函数

返回

  • (Array): 返回数组的切片

_.dropWhile(array, [predicate=_.identity])

创建一个数组切片，其中不包括从开始删除的元素。元素会被删除直到 predicate 的返回为 falsey（不再继续迭代）。predicate 有 3 个参数：value，index，array。

原数组不会被改变。
