lodashjs —— Array（二）

发表于 2019-05-15 | 分类于 代码

_.fill(array, value, [start=0], [end=array.length])

用从 start 开始到 end（但不包括 end）为止的值填充数组元素。

这个方法改变数组

_.findIndex(array, [predicate=_.identity], [fromIndex=0])

这个方法与 _.find 类似，不同的是 _.findIndex 返回 predicate 结果为 truthy 的第一个元素的索引，而不是元素本身。

_.findLastIndex(array, [predicate=_.identity], [fromIndex=array.length-1])

这个方法与 _.findIndex 类似，不同的是 _.findLastIndex 从右往左遍历元素。

_.head(array)

获取数组的第一个元素。

_.flatten(array)

将数组“压扁”一层

_.flattenDeep(array)

递归的“压扁”数组

_.flattenDepth(array, [depth=1])

递归的“压扁”数组到指定层数

_.fromPairs(pairs)

与 _.toPairs 相反；这个方法返回由键值对组成的对象。

_.head(array)

获取数组的第一个元素

别名：_.first

_.indexOf(array, value, [fromIndex=0])

获取数组中 value 第一次出现的索引（如果没有匹配的，就返回 -1），如果 fromIndex 是负数，返回 value 在数组末尾的偏移量。
