lodashjs —— Array（四）

发表于 2019-05-28 | 分类于 代码

_.pull(array, [values])

删除数组中所有与给定值相等的元素

与 _.without 不同，这个方法改变数组。通过 _.remove 移除元素。

_.pullAll(array, values)

这个方法与 _.pull 相似，不同的是它接收一个值的数组来移除。

_.pullAllBy(array, values, [iteratee=_.identity])

这个方法与 _.pullAll 相似，不同的是它接收一个被数组的每个元素和值调用的遍历器，来生成比较的标准。

_.pullAllWith(array, values, [comparator])

这个方法与 _.pullAll 相似，不同的是它接收一个用来比较数组中的元素和值的比较器。

与 _.differenceWith 不同，这个方法改变数组/

_.pullAt(array, [indexes])

删除数组中与索引对应的元素，并返回被删除的元素组成的数组。

与 _.at 不同，这个方法改变数组。

_.remove(array, [predicate=_.identity])

删除数组中所有执行 predicate 返回为 truthy 的元素，返回被删除的元素组成的数组。predicate 有三个参数：value, index, array。

与 _.filter 不同，这个方法改变数组。

_.reverse(array)

翻转数组。

这个方法改变数组，基于 Array#reverse。

_.slice(array, [start=0], [end=array.length])

创建一个从 start 开始到 end 结束（不包含 end）的数组切片。
